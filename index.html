HTML

  • The head element contains information about the webpage including charset, http-equiv, viewport
  • The body element represents the visible content shown to the user
  • The body element is Semantic as it describes the content and role on the page
  • Semantic elements are used for best practice and improved accessibily
  • The div element creates divisons in the page
  • & The span element creates inline division
  • But div and span are Non Semantic Elements
  • The style element defines CSS styles associated with the website
  • The base elements define base URL for webpage
  • The link elements connect external sources to HTML docs
  • The meta elements define metadata such as charset, author, description and viewpoint, as found in the head element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External CSS can be linked in the head section and universal changes can be made in that file
  • A card class can be added with the followin declarations. width, margin, border, box-shadow
  • Declarations may be set as follows:
  • width: 80%, margin: 40px auto; border: 5px solid gray; box-shadow: 5px 10px #888888;

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
